# Aethrix AI 聚合工坊 (Aethrix | 以太夜) - v1.0

Aethrix 是一个专为 AI 开发者、初创企业主和 AI 爱好者打造的高端 AI 工具聚合与导航平台。它不仅集成了海量的自动化工作流、提示词库和组件，还提供了极简且富有设计感的交互体验。

## 🌟 核心功能详解

### 1. AI 开发工坊 (AI Development Workshop)
这是平台的核心模块，集成了多种主流 AI 自动化方案：
*   **n8n 工作流**：内置 6000+ 精选工作流，涵盖公众号采集、自动化运营、数据分诊等实战场景。
*   **Dify 应用**：整合了丰富的 Dify DSL 模板，方便快速构建高性能 AI 应用程序。
*   **Coze 智能体**：精选多领域的 Coze Bot 配置方案，实现任务自动化。
*   **Skill 组件库**：提供结构化的 System Prompt 和 技能组件，点击即可复制使用。
*   **提示词工坊**：针对 Midjourney, Stable Diffusion, DALL-E 等模型优化的视觉化提示词库。

### 2. 一人公司资源 (Solopreneur)
专为独立开发者和“一人公司”设计的资源板块，提供产品变现方向、运营策略及工具推荐。

### 3. 教程与 API 集合
*   **教程/案例**：根据 AI 聊天、搜索、设计、写作等 16 个细分领域归纳的实战指南。
*   **API 市场**：收录官方及第三方聚合平台 API，帮您快速找到最实惠的算力支持。

### 4. 社区与互动
*   **AI 排行榜**：展示最受关注的 AI 模型与工具动态。
*   **匿名思想墙**：用户可以在此分享 AI 见解或寻求技术建议，支持管理员回复。

---

## 🛠 技术架构

*   **前台系统**：基于 HTML5 + Vanilla JS + CSS3 (Bureau F 极简主义设计风格)。
*   **后端服务**：
    *   **主站 (Node.js)**：负责用户管理、权限校验、静态资源渲染及数据接口。
    *   **微服务 (Python)**：用于各工坊模块（n8n, Dify, Coze, Skill）的数据解析与独立运行。
*   **数据持久化**：采用 JSON 文件存储方案（`data/` 目录），无需复杂数据库配置，开箱即用。

## 🚀 快速开始

### 1. 环境准备
确保您的系统中已安装：
*   **Node.js** (推荐 v18+)
*   **Python** (推荐 v3.10+)

### 2. 配置说明
复制目录下的 `.env.example` 并重命名为 `.env`，根据需要修改以下内容：
*   `ADMIN_PASSWORD`：管理员初始密码。
*   `PORT`：主站运行端口（默认 3006）。
*   `SMTP_HOST/USER/PASS`：用于发送邮箱验证码的邮件服务器配置。

### 3. 启动服务
在项目根目录下，双击运行以下脚本：
```bash
./start-all.bat
```
该脚本会自动启动主站及其他 6 个后端支撑服务。

### 4. 访问平台
启动成功后，在浏览器访问：
*   **主地址**：[http://localhost:3006](http://localhost:3006)
*   **管理员后台**：登录主站后，访问 `/admin.html`（需管理员权限）。

## 📂 项目目录说明
*   `public/`：前端静态页面与资源。
*   `data/`：存储站点配置、用户数据及思想墙内容。
*   `n8n/`, `dify/`, `coze/`, `skill/`, `prompt-workshop/`：各模块对应的资源及后台服务。
*   `server.js`：主站核心服务端代码。
*   `start-all.bat` / `stop-all.bat`：一键启动/停止所有服务的脚本。
*   `test-services.bat`：检查各端口服务运行状态的工具。
